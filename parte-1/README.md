# Asimov - Teste Técnico (Parte 1)

Este projeto representa a **Parte 1** do desafio técnico para a Asimov Academy. O objetivo foi transformar um design de alta fidelidade do Figma (Positivus) em uma landing page funcional, performática e responsiva utilizando tecnologias modernas de frontend.

### Ferramentas e Metodologia

- **Desenvolvimento:** O projeto foi construído utilizando **Next.js 16** e **Tailwind CSS 4**.
- **IA e Automação:** Foi utilizado o **Claude Code** integrado ao **Figma MCP** para acelerar a tradução dos tokens de design e a estrutura inicial dos componentes.
- **Animações:** Implementadas com **GSAP** para garantir transições suaves e interações modernas.
- **Carrosséis:** Utilização da biblioteca **Swiper** para a seção de depoimentos.

### Especificações de Design

O projeto segue rigorosamente as especificações do Figma:

- **Cores Principais:** Verde (`#B9FF66`), Escuro (`#191A23`) e Cinza (`#F3F3F3`).
- **Tipografia:** Utiliza a fonte **Space Grotesk** para todos os títulos e corpo de texto.
- **Componentes:** Inclui Navbar responsiva, Hero section, Grid de serviços, Accordion de processo de trabalho (Working Process) e Carrossel de depoimentos.

### Instruções para Rodar Localmente

1. **Navegue até a pasta do projeto:**
   ```bash
   cd parte-1
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

4. **Acesse no navegador:**
   O projeto estará disponível em [http://localhost:3000](http://localhost:3000).
