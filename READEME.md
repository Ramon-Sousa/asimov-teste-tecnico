# Asimov teste técnico

Este repositório contém a entrega do desafio técnico para a vaga de designer / frontend dev na Asimov academy. O projeto foi desenvolvido focando em fidelidade visual, performance e escalabilidade.

### Respostas ao teste (IA & Desenvolvimento)

- **Quais ferramentas de ia você usou?**
  Utilizei o Claude Code integrado ao Figma MCP para a parte-1 e o Antigravity (Gemini 3.1 Pro) também via Figma MCP para o suporte na parte-2.

- **Em que etapas a ia ajudou mais?**
  A ia foi essencial na interpretação técnica do protótipo de alta fidelidade do Figma. Ela permitiu transformar o design em um código limpo e modular de forma acelerada, garantindo que a estrutura seguisse boas práticas de desenvolvimento desde o início, visando a escalabilidade e a eficiência do projeto.

- **O que você ajustou manualmente e por quê?**
  Os ajustes manuais foram concentrados no refinamento da responsividade de elementos complexos, correção de imagens que apresentavam distorção (aspect-ratio) e a implementação de animações e microinterações para garantir uma experiência de usuário mais fluida e polida.

### Instruções para rodar localmente

Para baixar e configurar o repositório em sua máquina, siga os passos abaixo:

1. **Clone o repositório:**
   ```bash
   git clone [https://github.com/Ramon-Sousa/asimov-teste-tecnico.git](https://github.com/Ramon-Sousa/asimov-teste-tecnico.git)
   ```
2. **Instale as dependências (dentro de cada pasta):**
   Entre na pasta do projeto desejado e instale os pacotes:

   ```bash
   cd parte-1 # ou cd parte-2
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**

   ```bash
   npm run dev
   ```

4. **O projeto estará disponível em http://localhost:3000.**
