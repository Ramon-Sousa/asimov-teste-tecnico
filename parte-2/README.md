# Asimov - Teste Técnico (Parte 2)

Este repositório contém a **Parte 2** do desafio técnico para a Asimov Academy. Diferente da primeira etapa, este projeto focou no **processo criativo completo**, desde a concepção de wireframes até a implementação de uma interface voltada para a conversão e experiência de aprendizado.

### Processo Criativo e Design

O desenvolvimento desta etapa seguiu uma jornada de exploração de UI/UX:

- **Wireframing:** Foram criadas diferentes variações de wireframes para testar hierarquias de informação e fluxos de usuário.
- **Hero Section Estratégica:** A escolha final por uma **Hero com Mockup centralizado** foi estratégica. Como o objetivo da página é a venda de um curso, esse espaço foi projetado para ser versátil: atualmente exibe um mockup da plataforma, mas sua estrutura permite a fácil substituição por um **vídeo de apresentação**, elemento crucial para páginas de vendas de infoprodutos.
- **Identidade Visual:** Focada em um visual *premium*, utilizando o contraste do preto profundo com o azul acentuado (`#85E8EA`) e toques de laranja (`#FF6D0B`) para dar ênfase a elementos chave.

### Ferramentas e Metodologia

- **Desenvolvimento:** Construído com **Next.js 16** e **Tailwind CSS 4**.
- **IA e Automação:** Nesta etapa, utilizei o **Antigravity** como ferramenta aceleradora de código. O modelo de IA utilizado foi o **Gemini 3.1 Pro** em conjunto com o **Figma MCP**, o que permitiu uma integração fluida entre o design e a implementação técnica.
- **Responsividade:** O projeto foi refinado manualmente para garantir que a experiência mobile fosse tão impactante quanto a desktop, com inversão estratégica de elementos (Mockup vs Bullets) para melhor legibilidade.

### Instruções para Rodar Localmente

1. **Navegue até a pasta do projeto:**
   ```bash
   cd parte-2
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

4. **Acesse no navegador:**
   O projeto estará disponível em [http://localhost:3000](http://localhost:3000).
